## Primeiros Passos

Para começar a utilizar nossas APIs, você precisa:

1. [Criar uma conta](https://dashboard.byefive.com.br/signup) na plataforma ByeFive
2. Gerar suas credenciais de API no [painel de desenvolvedor](https://dashboard.byefive.com.br/developer)
3. Verificar a [autenticação](/api-reference/authentication) necessária para as chamadas de API

## Ambientes Disponíveis

A ByeFive oferece três ambientes para desenvolvimento e uso das APIs:

| Ambiente | Base URL | Descrição |
|----------|----------|-----------|
| Desenvolvimento | `https://{api}.app.dev.byefive.com.br/rest` | Para testes durante o desenvolvimento |
| Homologação | `https://{api}.app.staging.byefive.com.br/rest` | Para validar integrações antes da produção |
| Produção | `https://{api}.app.byefive.com.br/rest` | Ambiente de produção para uso em sistemas reais |

Substitua `{api}` pelo nome da API específica que você está utilizando (ex: `qrcode`, `sign`, etc.).

<Note>
  As APIs são constantemente atualizadas com novos recursos. Consulte a documentação específica de cada API para informações detalhadas sobre os endpoints disponíveis.
</Note>
